About this project

Built by Jay Manamperi as a portfolio project demonstrating a complete serverless AWS architecture.

Paste any long URL into the box above and get a short, shareable link instantly. Short links redirect to the original URL via a fast AWS Lambda function.

Tech stack

  • Frontend — Static HTML/CSS/JS served from S3 via CloudFront

  • API — AWS API Gateway HTTP API

  • Backend — Python 3.11 Lambda functions running as Docker images

  • Database — DynamoDB for URL mappings

  • CDN — CloudFront with custom domain and SSL certificate

  • Container registry — Amazon ECR

  • Infrastructure — Terraform with S3 remote state

  • CI/CD — GitHub Actions with OIDC authentication — no static AWS credentials



Source code available on GitHub:

  • aws-terraform-bootstrap — Shared AWS bootstrap (OIDC role, S3 state bucket)

  • shorten-url-infra — AWS infrastructure (VPC, API Gateway, Lambda, DynamoDB, CloudFront)

  • shorten-url-app — Application code (Python Lambda functions, Docker images, frontend)
